================================================================
Apple領収書 自動コレクター セットアップ手順
================================================================

このツールを使うには、初回だけGoogle Cloud Consoleで
OAuth設定が必要です。10分程度で終わります。
画面を見ながら順番に進めてください。

----------------------------------------------------------------
【ステップ1】 Google Cloud Console にログイン
----------------------------------------------------------------
1. ブラウザで https://console.cloud.google.com/ を開く
2. eightmedia0808@gmail.com でログイン

----------------------------------------------------------------
【ステップ2】 新しいプロジェクトを作成
----------------------------------------------------------------
1. 画面上部の「プロジェクトを選択」ドロップダウンをクリック
2. 「新しいプロジェクト」をクリック
3. プロジェクト名: Apple Receipt Collector
4. 「作成」ボタンをクリック
5. 作成完了後、画面上部のプロジェクト選択で
   "Apple Receipt Collector" が選択されていることを確認

----------------------------------------------------------------
【ステップ3】 必要なAPIを有効化
----------------------------------------------------------------
1. 左メニュー → 「APIとサービス」 → 「ライブラリ」
2. 検索欄に "Gmail API" と入力
3. 「Gmail API」をクリック → 「有効にする」ボタン
4. 戻って、検索欄に "Google Drive API" と入力
5. 「Google Drive API」をクリック → 「有効にする」ボタン

----------------------------------------------------------------
【ステップ4】 OAuth 同意画面を設定
----------------------------------------------------------------
1. 左メニュー → 「APIとサービス」 → 「OAuth同意画面」
2. User Type: 「外部」を選択 → 「作成」
3. アプリ情報を入力:
   - アプリ名: Apple Receipt Collector
   - ユーザーサポートメール: eightmedia0808@gmail.com
   - デベロッパーの連絡先情報: eightmedia0808@gmail.com
4. 「保存して次へ」
5. スコープ画面: そのまま「保存して次へ」
   （アプリ側でスコープを指定するのでここでは何も追加しなくてOK）
6. テストユーザー画面:
   - 「+ ADD USERS」をクリック
   - 領収書メールが届く全Gmailアドレスを追加
     （後から追加もできます）
   - 「保存して次へ」
7. 「ダッシュボードに戻る」

----------------------------------------------------------------
【ステップ5】 OAuthクライアントIDを作成
----------------------------------------------------------------
1. 左メニュー → 「APIとサービス」 → 「認証情報」
2. 上部の「+ 認証情報を作成」 → 「OAuth クライアントID」
3. アプリケーションの種類: 「ウェブアプリケーション」
4. 名前: Apple Receipt Collector Local
5. 「承認済みのリダイレクトURI」の「+ URIを追加」を2回クリックして
   以下の2つを追加（コピペでOK）:

   http://localhost:8765/oauth/drive/callback
   http://localhost:8765/oauth/gmail/callback

6. 「作成」をクリック
7. ポップアップが出たら「JSONをダウンロード」をクリック

----------------------------------------------------------------
【ステップ6】 ダウンロードしたJSONを設置
----------------------------------------------------------------
1. ダウンロードした client_secret_xxxxx.json ファイルを
2. 以下の場所に "credentials.json" という名前で保存:

   /Users/risa/apple-receipt-collector/credentials.json

   ターミナルでやる場合（Downloadsから移動・リネーム）:
   mv ~/Downloads/client_secret_*.json /Users/risa/apple-receipt-collector/credentials.json

----------------------------------------------------------------
【ステップ7】 自動起動設定（launchd登録）
----------------------------------------------------------------
ターミナルで以下を1行ずつ実行:

   cp /Users/risa/apple-receipt-collector/launchd/com.eightmedia.apple-receipt-ui.plist ~/Library/LaunchAgents/
   cp /Users/risa/apple-receipt-collector/launchd/com.eightmedia.apple-receipt-sync.plist ~/Library/LaunchAgents/
   launchctl load ~/Library/LaunchAgents/com.eightmedia.apple-receipt-ui.plist
   launchctl load ~/Library/LaunchAgents/com.eightmedia.apple-receipt-sync.plist

----------------------------------------------------------------
【ステップ8】 WebUIを開いてアカウント設定
----------------------------------------------------------------
1. ブラウザで http://localhost:8765 を開く
2. 「Driveアカウントを設定」ボタンをクリック
   → eightmedia0808@gmail.com でログイン → 権限を許可
3. 「+ Gmailアカウントを追加」ボタンをクリック
   → 領収書を受信するGmailアドレスでログイン → 権限を許可
   ※ 必要なアカウントの数だけ繰り返す
4. 「🔄 今すぐ同期」ボタンで初回同期を実行
   → 過去メール分のPDFも全件Driveに保存されます

----------------------------------------------------------------
完了 ! あとは1時間ごとに自動で新着領収書が保存されます。

【新しいGmailアドレスを追加するには】
  http://localhost:8765 を開いて「+ Gmailアカウントを追加」だけ
  ※ 新規アドレスは事前に「OAuth同意画面 → テストユーザー」に
     追加しておく必要があります（ステップ4-6）
     アプリを「公開」にすれば不要になります（審査必要なため
     当面はテストユーザー方式推奨）

【困ったとき】
  ログ確認:
    /Users/risa/apple-receipt-collector/logs/ui.log
    /Users/risa/apple-receipt-collector/logs/sync.log

  再起動:
    launchctl unload ~/Library/LaunchAgents/com.eightmedia.apple-receipt-ui.plist
    launchctl load ~/Library/LaunchAgents/com.eightmedia.apple-receipt-ui.plist
